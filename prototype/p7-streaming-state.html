P7 Design Rules:

• ghost bar = model color 30% + dashed right border + pulse animation. Width updated 1fps via rAF.

• completion fires immediately on SSE message_stop, bypasses rAF throttle. 200ms CSS transition to solid.

• 🧠 thinking shown after 3s with no content_block_start. Replaced by first tool name on arrival.

• sparkline gap — no estimate, no fake data. Gap = honest "pending."

• All animations use opacity/transform only (compositor layer, no reflow). ~25 LOC CSS + ~15 LOC JS.
