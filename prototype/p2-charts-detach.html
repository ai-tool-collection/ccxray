P2 Design Rule:

• Charts move from Agent Card → Steps Panel sticky header. Agent Card = text-only summary.

• Chart width = Steps Panel width (400-800px). min-width: turnCount × 2px with overflow-x: auto fallback.

• Blue cursor line is vertical through all 3 charts + step list. Click chart → select turn. Click step → move cursor.

• Agent Card keeps: header, context/cache/cost TEXT, nav items. No charts. Cleaner, less scroll.

• Lane Label ↔ Agent Card alignment preserved (both 240px text). Charts don't need to match lane labels.
