P9 Design Rules:

• All touch adaptations behind @media (pointer: coarse) or touchstart detection. Desktop = zero impact.

• Resize handle: visual 16px + ::before transparent 44px hit zone (Apple HIG compliant).

• Overview bar edge: lights up 3px #58a6ff on touch-down (200ms). Replaces cursor affordance.

• Brush-to-zoom: visible 🔍 button (not hidden long-press). Tap → brush mode → drag → release.

• Snap-to-nearest: 22px radius, O(log n) binary search. Works on turn bars + chart bars.

• Direction lock: 6px dead zone (not 10px). { passive: false } on touchmove listener.

• Pinch-to-zoom: JS Math.hypot on two-finger delta. touch-action: pan-y on SVGs.

• iPad Magic Keyboard → pointer: fine → desktop experience (correct behavior).

• No navigator.vibrate() (not supported in Safari).

• ~120 LOC total (100 JS + 20 CSS).
